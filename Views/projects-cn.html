虚拟现实阅读界面的设计实现和测试

七月 2018 - 十一月 2018, 墨尔本

项目设计并实现了一个在虚拟环境中的阅读界面，该界面不同于传统的纸质阅读和电子书阅读，它是一个拥有沉浸式体验的全景虚拟阅读环境，并引入了声音、触觉效果来制造一种“身临其境”的感觉。该环境基于PC端搭建，使用了Oculus Rift头戴显示设备。

在首版界面搭建完毕后，本项目随即邀请了12位用户进行了一个用户研究，用以探查该界面的设计缺陷以及评估该界面的用户友好程度。经研究发现，即使该界面仍有待完善，用户对于新型的阅读方式依然抱有有极大的兴趣。因此我们的研究结论是，虚拟阅读的应用，仍旧是一个有极大探索空间的领域。

相关技能: Unity, C#, Oculus Rift, 原型设计, 用户可用性研究

项目督察员: Tilman Dingler, Niels Van Berkel, Ebrahim Babaei

SortingShop: 社交应用的界面设计

七月 2018 - 十一月 2018 墨尔本

该项目设计并构建了一个远程协作众包平台的界面原型设计，该平台能够让用户协同完成组内工作。

项目灵感来自于亲和图法，该方法被广泛用于分析用户需求，且常见于使用便利贴和白板来进行组内讨论。

考虑到远程协作和环境友好的设计目的，我们设计提出了SortingShop，一个能够让多个用户同时集思广益/头脑风暴的平台，它允许用户基于一个核心话题提交自己的想法/创意，然后将这些提交汇集起来，让每一个用户都能用自己的分类方法进行分类整理，最终将所有人的分类结果整合，分析出一定的规律/趋势，以供组内后续讨论。

SortingShop的独特性在于，它公平地考虑所有组内用户的想法和分类方法，并且整合所有用户的分类集合，将它们以韦恩图的形式展示出来，其中还包括有争议的、未分类的提交。

相关技能: 原型设计, Sketch, Adobe XD, 用户可用性研究

合作人: 王潮, 王魏

基于SparkPeople的用户体验评估

四月 2018 - 六月 2018, 墨尔本

该项目计划并实施了一个基于用户的可用性研究，研究对象是Sparkpeople.com网站，一共随机选取了14位新用户进行了研究，其中四场在墨尔本大学交互设计实验室（提供视线追踪装置）进行，另外十场在普通会议室进行。

该研究旨在针对SparkPeople网站的关键功能来进行评估，而这些关键功能旨在帮助网站用户保持身体和饮食健康。研究人员提供具体任务给用户，用户需要使用网站的关键功能来完成这些任务，而研究人员将观察用户完成这些任务的情况，以此识别其设计上的缺陷和可改进之处。

除了评估每个功能的用户可用性之外，该研究仅将目标用户集中在SparkPeople的新用户上，并且了解他们继续使用这个网站的可能性。研究过程中包括一系列的定性和定量测量，例如操作的错误率和成功率，完成任务的时间，用户完成任务期间的点击次数和打开页面的数量，用户的反馈以及满意情况

相关技能: 用户可用性研究, 用户访问, 启发式评估

合作人: Kirsty Fromholtz, 汪莹, Aneesh Rajiv Sawant

EOS - Web应用的设计开发

二月 2018 - 六月 2018, 墨尔本

EOS是一个基于网页的应用，它旨在帮助高等教育学府的学生为未来职业道路做好准备，增进雇主与未来潜在雇员的互动机会和相互理解。

By using HCI processes design methods, this project has designed and developed a full-stack web application to address an openIDEO challenge: “How might we better prepare all learners for the need of tomorrow by reimagining higher education”.

通过使用HCI流程设计方法（采集用户需求，整理分析用户痛点，设计并实现产品，用户测试和产品改进升级），该项目设计并开发了一个全栈式Web应用程序，以解决发布在openIDEO上的挑战之：“我们如何通过重新构想高等教育，以此来更好地为所有学习者做好准备，以满足未来的需求”。

相关技能: HTML/CSS, Javascript, Node.js, Express, MongoDB, Gith, Heroku

合作人: 王子禾, 梁嘉慧
