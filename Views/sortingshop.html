SortingShop

This project has designed and implemented an affinity diagramming APP interface, which is meant to help the traditional brainstorming and diagramming processes more effective.

Inspired by a workshop of fieldwork for design that conduct in the University of Melbourne, our group find that nowadays, brainstorming meetings tend to use post-notes and white board to gather and categorize ideas. This method has been widely taken into pratice in real-world design companies and institutions, but it requires co-located colleagues and members' unbiased judgements, which is not easy to apply in a real group work.

Therefore, we proposed SortingShop, an online tool that conducts the same functionality as brainstorming and affinity diagramming, but allows remote meeting work and provides unbiased diagramming system. It's easy to start and get familiar and all the meeting result can be saved online and reviewed at any time.

SortingShop is an application that can help co-working groups manage their ideas effectively and comprehensively. Multiple login channel provided.

User can choose to start a new discussion or join a current one, depends on what kind of role our user plays in his/her group.

If starting a new group, the following information will needed:

  • Group name
  • Discussion topic
  • Invite members

    • The list of group members can be accessed from the SortingShop colleague list, or use code to invite people when the discussion was created.

    After the discussion group was created, use the group code to invite members.

    Otherwise, use the code to join a current discussion group, easy and done.

    Before any discussion happens, ideas must be gathered. Users can post his/her own ideas or proposals onto the idea pool, which can be seen by all the members in the group.

    Based on certain principles, our used can drag ideas from the idea pool and categorize them.

    Or re-categorize some ideas from different categories.

    SortingShop gathers all the membersâ€™ categories and represent all the categories as a individual or overlapping circles, where the group can see different ideas of categorization.

    The group can continue on further discussion and categorize those controversial ideas.

    Categories can be modified at any time.

    User can check all the projects that he/she has participated from the user profile, including details of the group and the result.
